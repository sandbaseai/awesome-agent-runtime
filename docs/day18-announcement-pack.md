## Positioning

```text
Awesome Agent Runtime is a curated map of 500 projects across the production AI agent infrastructure stack.
```

The announcement should emphasize:

- production agents need infrastructure, not just prompts
- the stack spans runtimes, sandboxes, browser automation, tool protocols, memory, evals, observability, model gateways, and deployment
- the repository is useful to builders even if they never use SandBase
- maintainers are welcome to submit corrections or missing projects

## Do Not Use

Avoid language like:

- "the definitive list"
- "the best 500 projects"
- "we ranked the ecosystem"
- "SandBase powers all of these"

The list is a curated landscape, not a ranking or endorsement.
